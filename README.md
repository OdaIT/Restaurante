**Tema escolhido:** O Menu Digital (Restaurante Copo Cheio)

**Link do repositório GitHub:** https://github.com/OdaIT/Restaurante.git

**Passos para executar a vossa página:** 
- Os submenus são foldable (clicar no nome dos mesmos);
- Os artigos são flip cards (passar o rato por cima);
- Testar interatividade do calendario com a submissao do form;

**Principais decisões tomadas e justificação da adequação:** 
Nós queríamos que a página fosse o mais realista possível, que o calendário tivesse completamente funcional, o que se revelou um problema complexo mesmo com a assistência da AI. O calendário só permite reservas em datas today ou mais, não permite mais de x reservas por dia, permite ao estabelecimento definir que dias estão lock, por semana(para folga) ou data especifica(feriados etc).
Os filters também foram um grande desafio tendo inicialmente começado por fazê-los através de um array em JS, com o menu a constar todo em uma classe com o nome do prato, description e 'alergênicos' tendo depois restruturado toda a secção do menu por uma questão de simplificação.
